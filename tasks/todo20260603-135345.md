# PR #73 review comment follow-ups (Issue #37)

PR: https://github.com/moisesja/net-did/pull/73

## Review comments to address

1. **Container immutability for `Artifacts["log.entries"]`** — `LogEntrySerializer.ParseJsonLines` returns the concrete `List<LogEntry>` typed as `IReadOnlyList<LogEntry>`; a consumer can downcast and mutate. Records make entries immutable; harden the container too. Also drop the redundant `(IReadOnlyList<LogEntry>)entries` cast.
2. **Magic-string artifact keys** — `"did.jsonl"`, `"did.json"`, `"did-witness.json"`, `"log.entries"` are duplicated across Create/Update/Deactivate/Resolve and tests. Hoist to a public constants class so consumers (notably `net-wallet-mcp`) bind to symbols rather than re-typing literals.
3. **Record equality shallow over `Proof`** — informational only; no code change required.
